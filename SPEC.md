## Annex 1: Details of Berger Table

The round-robin pairing system uses Berger tables to determine pairings for each
round. In a tournament with N players (N even), there are N-1 rounds. Each
player plays every other player exactly once.

The Berger table algorithm:

- Player N is fixed in position
- Players 1 through N-1 rotate clockwise each round
- In each round, players in corresponding positions are paired
- Colors alternate: odd-numbered rounds have the fixed player as white,
  even-numbered rounds as black
